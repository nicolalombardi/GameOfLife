# GameOfLife
Game of Life implementation written in Java using libGDX.
## Features:
- Cells "painting" (drag your mouse around to set cells to the alive or dead state)
- Configurable grid size
- Configurable simulation speed
- Included patterns
- Looks good!

## Run it
Either download the .jar file in the releases section or compile the gradle project from the sources.
